# Proyecto Login Practica

Este proyecto es una aplicación de ejemplo para practicar Git y el uso de variables de entorno en el desarrollo de aplicaciones. Para configurar el proyecto, primero clona el repositorio con `git clone <URL_DEL_REPOSITORIO>` y entra a la carpeta del proyecto con `cd login-practica`. Luego, instala las dependencias con `npm install`. Crea un archivo llamado `.env` en la raíz del proyecto con las variables de entorno necesarias, por ejemplo: `PORT=3000`, `DB_USER=usuario`, `DB_PASSWORD=contraseña` y `SECRET_KEY=clave_secreta`, ajustando los valores según tu entorno local. Para ejecutar la aplicación, usa `npm start` y accede a `http://localhost:3000` o al puerto definido en `.env`. Las variables de entorno permiten configurar la aplicación sin modificar el código fuente; `PORT` define el puerto, `DB_USER` y `DB_PASSWORD` se usan para la base de datos, y `SECRET_KEY` se puede usar para tokens o contraseñas. Recuerda nunca subir tu archivo `.env` al repositorio público. Después de hacer cambios en el proyecto, agrega los archivos con `git add .`, haz un commit con `git commit -m "Descripción del cambio"` y sube tus cambios con `git push origin <nombre_branch>`.
